# Dennis Kibathi | Personal Portfolio

A clean, responsive, and interactive personal portfolio website designed to showcase web development training progress, technical skills, and active projects.

## 🚀 Features

*   **Interactive UI:** Includes a dynamic JavaScript-powered button that greets visitors instantly upon clicking.
*   **Modern Design:** Styled with a clean, responsive layout using custom CSS variables, soft shadows, and a green color palette.
*   **Structured Sections:** Organized into logical modules for an About profile, Skills inventory, and Project display.

## 🛠️ Built With

*   **HTML5:** Semantic document structure (`<header>`, `<section>`, `<footer>`).
*   **CSS3:** Custom layouts, responsive margins, hover transitions, and rounded card styling.
*   **JavaScript (ES6):** DOM manipulation and event listening for live user interaction.

## 💻 How to Run Locally

1. Clone or download this project folder onto your local computer.
2. Double-click the `index.html` file to open it directly in any web browser.
3. Open the project folder in **VS Code** and use the **Live Server** extension for real-time development tracking.
